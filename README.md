# Aduanas Chile - NewCooltura Informada

Modulo de informacion aduanera para Chile. Parte del ecosistema NewCool.

## Funcionalidades

- **Buscador de Aduanas**: 17 aduanas y pasos fronterizos con filtros por region y tipo
- **Calculadora de Aranceles**: Calcula impuestos de importacion (arancel 6%, IVA 19%, lujo 15%)
- **Franquicias**: Informacion sobre exenciones (viajeros, courier, zonas francas)
- **Productos Prohibidos/Restringidos**: Lista de restricciones de ingreso
- **Tramites Aduaneros**: DIN, DUS, admision temporal, transito
- **Glosario**: Terminos aduaneros explicados

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion

## Desarrollo

```bash
npm install
npm run dev
```
